##用css控制段落

```
<style>
	yourtexID {
		text-indent:40px;//段首缩进	text-text-align:center;//文字对齐方式，有left，right可选
		text-decoration:underline dotted red;//加下划线
		text-shadow:10px 10px 10px red;//设置文字阴影 这四个参数分别代表 水平方向偏移 垂直方向偏移 模糊值 阴影颜色 后两个参数为可选参数
	}
</style>
```
